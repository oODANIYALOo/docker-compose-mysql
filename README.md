# docker-compose-mysql
mysql phpmyadmin docker compose install

### first install docker and docker compose
[click me to install docker](https://docs.docker.com/engine/install/)

### download project
`git clone https://github.com/oODANIYALOo/docker-compose-mysql.git`

### goto project path
`cd docker-compose-mysql`
### run project with
`docker compose up -d`
### stop project with
`docker compose stop`
